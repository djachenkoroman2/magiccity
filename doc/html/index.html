Документация MagicCity

MagicCity — процедурный генератор городских point clouds. citygen строит детерминированные городские тайлы из YAML-конфигов и экспортирует ASCII PLY вместе с JSON-файлом metadata.

Документация описывает runtime-модели, конфиги, дороги, биомы, parcels, здания, sampling, catalogs и metadata. Координатная конвенция проекта: горизонтальная плоскость x/y, высота z.

Разделы

  • Справочник по конфигурации — YAML-схема, значения по умолчанию, валидация и примеры.
  • Дороги — road primitives, модели дорог, road profiles и surface-классы.
  • Биомы — urban_fields, классификация районов и влияние биомов на дороги/здания.
  • Parcels — blocks/parcels, oriented subdivision, buildable areas и placement зданий.
  • Footprints зданий — поддержанные footprint types, aliases, геометрия и clearance.
  • Крыши зданий — roof types, aliases, функции высоты и sampling.
  • Генерируемые объекты — object feature ids, semantic classes и metadata counts.
  • Worldgen catalogs — catalogs, стадии пайплайна и правила добавления ids.
  • Universal showcase — большой демонстрационный сценарий и поля metadata для проверки.
